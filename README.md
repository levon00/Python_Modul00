# Python_Module00
*This project has been created as part of the 42 curriculum by lkhalaty.*

---

## Description

Python_Module00 is an introductory Python project focused on learning the fundamental syntax and core programming concepts of Python through community garden themed exercises.

The project introduces variables, input/output, conditions, loops, functions, recursion, and type annotations using simple real-world examples related to garden management and harvest tracking.

---

## Project Goal

- Learn basic Python syntax and semantics
- Understand functions and control flow
- Practice working with variables and expressions
- Learn iterative and recursive approaches
- Discover Python type annotations
- Write clean code following flake8 standards

---

## Instructions

### Requirements

- Python 3.10+
- flake8

### Running Exercises

Each exercise contains only one function inside its own `.py` file.

To test exercises:

```bash
python3 main.py
```

The provided `main.py` helper imports and tests your functions automatically.

### flake8 Check

To verify code style:

```bash
python3 -m flake8 .
```

---

## General Rules

- Each exercise must be inside its own folder
- Each file must contain only the requested function
- No main program
- Do not use:

```python
if __name__ == "__main__":
```

- Function names must match exactly
- Undefined behavior for invalid or negative inputs
- Type hints required in Exercise 7
